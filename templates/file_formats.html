Para que se faça uso da plataforma e dos seus diversos algoritmos é necessário que seja realizado inicialmente o upload dos dados sobre os quais serão realizadas as análises. A entrada de dados na plataforma é realizada apenas através de arquivos .csv e é importante que sejam passados os parâmetros corretos na leitura dos arquivos para garantir o bom funcionamento das funções. Entenda mais sobre a estrutura do arquivo e os parâmetros de entrada nos tópicos a seguir.

Parâmetros de entrada


Separador

O separador é o símbolo empregado para separar os dados de diferentes colunas em um arquivo .csv. O separador mais comum (principalmente em arquivos na língua inglesa) é a vírgula (,) mas em muitos arquivos em português emprega-se o ponto-e-vírgula (;).

Coluna de Datas

O parâmetro "Coluna de datas" é utilizado para identificar qual das colunas do seu arquivo encontram-se os dados de tempo (caso exista uma coluna para tal).

Cabeçalho

Assinale a opção "Cabeçalho" em caso de seu arquivo possuir um cabeçalho, ou seja, no caso da primeira linha do seu arquivo .csv conter informações que não são dados a serem analisados, mas sim os títulos das colunas.

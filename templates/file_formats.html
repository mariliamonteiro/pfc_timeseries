Sobre o Projeto

A Plataforma de Análise de Séries Temporais faz parte de um dos Projetos de Fim de Curso desenvolvidos na Seção de Engenharia Cartográfica (SE/6) do Instituto Militar de Engenharia no ano de 2018.

O projeto faz parte do portifólio que busca desenvolver ferramentas de ajustamento e análise de dados empregando a linguagem de programaçao Python, com propósitos principalmente acadêmicos.

Para saber mais sobre o Instituto Militar de Engenharia (IME), acesse o Site do Instituto

Código Aberto

Uma das premissas do projeto foi que ele tivesse código aberto, de modo a poder ser editado mesmo depois da graduação de seus autores originais, permitindo melhorias constantes no sistema.

O código do projeto encontra-se disponível no GitHub

Tendo sido desenvolvida em Python 3, a plataforma foi construida empregando Flask e os modelos aqui utilizados se baseiam principalmente naqueles disponibilizados pelo StatsModels.
